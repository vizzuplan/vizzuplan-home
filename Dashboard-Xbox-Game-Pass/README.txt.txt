# 📊 Dashboard Xbox Game Pass Subscriptions

Visual dashboard para análise de assinaturas do Xbox Game Pass com foco em planos, serviços adicionais e status de renovação.

## 📁 Estrutura do Projeto

- `DashboardXbox.xlsx`: planilha principal com dados, cálculos e layout visual
- `assets/`: paleta de cores oficial e logos aplicáveis (Xbox/VizzuPlan)
- `README.md`: documentação base do projeto

## 🎯 Objetivo

Oferecer uma visualização clara e rápida sobre:
- Distribuição de assinantes por plano (`Core`, `Standard`, `Ultimate`)
- Valor total gerado por serviços (EA Play, Minecraft Season Pass)
- Status de auto-renovação e comportamento por perfil

## 🧩 Funcionalidades

- Indicadores semafóricos para performance por assinatura
- KPIs visuais com aplicação de paleta Xbox
- Layout organizado por seções: visão geral, serviços extras, comportamento de assinatura

## 🖼️ Visual

O dashboard foi criado com design limpo, uso das cores:
- `#9BC848` e `#22C55E` para indicadores positivos
- `#E8E6E9` para fundo neutro
- Barra de progresso em valor total + gráficos por plano

## 📌 Próximas Evoluções

- Inserção de filtros interativos na planilha
- Automação com fórmulas de performance por assinatura
- Exportação para versões online (Power BI, Google Data Studio)
